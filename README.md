# 🚀 AI Interview Assistant

A data-driven interview preparation system that analyzes problem-solving patterns, detects weaknesses, and simulates real interview scenarios using adaptive logic.

---

## 🌐 Live Demo

👉 [Open App](https://ai-interview-assistant01.streamlit.app/)

---

## 🧠 What This Project Does

This is not just a tracker — it is an **intelligent system** that:

* Tracks solved coding problems
* Analyzes topic-wise performance
* Detects weak areas automatically
* Recommends what to practice next
* Simulates real interview scenarios
* Calculates interview readiness score

---

## ⚙️ Core Features

### 📊 Performance Analytics

* Topic-wise accuracy calculation
* Progress tracking with visualization
* Weak topic detection

### 🎯 Recommendation Engine

* Suggests next topics based on performance
* Identifies improvement areas

### 🧠 Pattern Intelligence

* Detects strong vs weak topics
* Highlights over-focused areas

### 🧪 Adaptive Interview Mode

* Automatically selects weak topics
* Generates practice questions
* Includes timer-based simulation

### 📈 Interview Readiness Score

* Combines accuracy, coverage, and balance
* Gives real-time preparation level

## 🚀 How It Works

1. Add or import solved problems
2. System analyzes performance
3. Weak topics are identified
4. Recommendations are generated
5. Adaptive interview mode tests your skills
